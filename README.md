# jwp-chess
웹 기반 체스 게임 구현을 위한 저장소

## 미션 회고 및 학습 내용

### 미션 포인트 ⏰ 
   - Spring Data JDBC의 간략한 사용법을 파악한다.
   - DAO를 Repository로 변경하는 방법을 학습한다.
   - 레이어별 역할을 명확히 한다.
### 학습 내용 📖
   - Spring Data JDBC
      - ID를 사용한 Mapping
      - @Modifying을 통해 업데이트가 가능하다.
### 어려웠던 내용 😂
